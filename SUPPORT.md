# Support

Thank you for using **I2P-Style Pseudocode**! 🎓

If you need help, have a question, or want to report a problem, please use GitHub so everything stays organized and visible to the community.

---

## 🐛 Report a Bug (Issues)

Use **Issues** if something is not working as expected.

**Examples:**
- Syntax highlighting is incorrect or missing
- A keyword is not recognized
- The extension behaves differently than documented

👉 Create an issue:  
https://github.com/i2p-hub/i2p-pseudocode/issues

**Before submitting:**
- Check if your issue has already been reported
- Include a short example of your pseudocode
- Add screenshots if helpful

---

## 💡 Ask Questions or Share Ideas (Discussions)

Use **Discussions** for general questions, ideas, or help using the extension.

**Examples:**
- “How do I format functions in I2P pseudocode?”
- Suggestions for new features or improvements
- Clarification on pseudocode conventions

👉 Start a discussion:  
https://github.com/i2p-hub/i2p-pseudocode/discussions

---

## 🚀 Feature Requests

If you have an idea to improve the extension:
- Use **Discussions** for early ideas or feedback
- Use **Issues** if you have a clear, actionable request

---

## 🤝 Contributing

Interested in contributing?  
Feel free to open an issue or discussion to get started.
